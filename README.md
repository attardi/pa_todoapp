pa_todoapp
==========

This repo contains the nodejs app showed at the course Advanced Programming @ University of Pisa.
It's just a simple TODO list implemented in nodejs + mongodb + jQuery.

Installation
============

    npm install

Use
===
Create directory for mongodb:

    mkdir db
    mongod --dbpath db

Start the app:

    node app.js

Open the browser at:

    http://localhost:8080
